## Dependencies

- Node.js
- Express
- EJS
- bcrypt
- body-parser
- cookie-session

## Getting Started

- Install all dependencies (using the `npm install` command).
- Start the web server with the `npm start` command.
- Browse to `http://localhost:8080/` 
- If the defualy port 8080 

## Customization
- Change the defualt port in `express_server.js`
